🎯 Kya hoga: Aapka data automatically Google Sheets mein save hoga. Koi bhi committee member kabhi bhi phone se dekh sakta hai.
Step 1: script.google.com kholein (Google account se login karen)
Step 2: "New Project" click karen
Step 3: Neeche diya hua code paste karen aur Save karen
Step 4: "Deploy" → "New Deployment" → "Web App" → "Anyone" → Deploy
Step 5: Jo URL mile, woh upar "Setup" tab mein paste karen

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.openById(data.sheetId || createSheet());
  const sheet = ss.getSheetByName(data.tab) || ss.insertSheet(data.tab);
  if(data.clear) sheet.clearContents();
  if(data.headers) sheet.getRange(1,1,1,data.headers.length).setValues([data.headers]);
  if(data.rows && data.rows.length) {
    const last = sheet.getLastRow();
    sheet.getRange(last+1,1,data.rows.length,data.rows[0].length).setValues(data.rows);
  }
  return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON);
}
function createSheet() {
  const ss = SpreadsheetApp.create("Jama Masjid Bhagwanpur — Data");
  return ss.getId();
}
⚠️ Note: "Sheet ID" apni Google Sheet ke URL se milega (docs.google.com/spreadsheets/d/SHEET_ID/edit). Isko Google Apps Script mein openById() mein daalein.
